# 配置分析

## 目标
分析哪些开关控制系统行为，理解配置体系。

## 分析要求

1. 找出所有配置入口：环境变量、配置文件、命令行参数、远程配置
2. 说明每个配置控制什么行为
3. 标出默认值、优先级、覆盖关系
4. 区分开发/测试/生产配置
5. 找出最影响系统行为的 5 个配置项
